- **Interactive Cursor Tracking**: Smooth exponential moving average (EMA) tracking with setpoint and mouse cursors
- **Dynamic Boid Simulation**: 25 flocking particles with realistic behaviors (separation, alignment, cohesion, and attraction to cursor)
- **Network Visualization**: Connects each boid to its 3 nearest neighbors with glowing green lines
- **Adaptive Mouse Cursor**: Changes size based on distance to setpoint (11-51px range)
- **Navigation Morphing**: Mouse cursor morphs to match navigation item bounding boxes on hover
- **Matrix Aesthetic**: Black background with green accents and long particle trails
